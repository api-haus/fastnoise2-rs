FastNoise2 WASM Demo (Emscripten)

About This Demo

This demonstrates FastNoise2 C++ library compiled to WASM using Emscripten and linked with Rust.

Key Technologies:

  • FastNoise2 (C++) + fastnoise2-rs (Rust FFI bindings)
  • Target: wasm32-unknown-emscripten
  • No wasm-bindgen - uses native Emscripten JS glue
  • WASM SIMD128 enabled for performance
